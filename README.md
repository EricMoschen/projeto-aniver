# Projeto Aniver

Carta de aniversário interativa com animação de envelope, efeito de digitação e elementos visuais românticos.

## Como executar

Abra o arquivo `index.html` no navegador.

## Estrutura

- `index.html`: estrutura da página.
- `styles.css`: estilos e animações.
- `scripts.js`: comportamento interativo e mensagem digitada.
- `ursinhos.gif`: imagem usada dentro da carta.
